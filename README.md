# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- リモートはネットワーク上にある共有可能でバックアップ等を利用する領域
- ローカルは自らのPC上の作業領域

## プッシュとマージの違いは何でしょうか？

- プッシュはデータのアップロードを指し、マージはブランチで分けた作業の合流、統合を指す

## コミットとプッシュの違い

- コミットは履歴の保存を指す、プッシュはアップロードを指す

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- どんな編集を行なったのかを示すメッセージでチーム独自のルール等を守るのが最適

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- ローカルは個人ですが、プルの場合は他者が確認するためバグの早期発見が可能になる

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- 自分以外が描いたソースコードを把握するため、書いた人と相談しながら作業を進めるべき
